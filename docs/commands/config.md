This command allows you to init fleex.

Usage:

| Flag | Name          | Description                             | Default |
| ---- | ------------- | --------------------------------------- | ------- |
| `-o` | `--overwrite` | If the fleex folder exists overwrite it |         |
| `-u` | `--url`       | Config folder url                       |         |

!!! warning
    Fleex will initialize all configuration files in `$HOME/fleex/`

Examples:
```
fleex config init
```
